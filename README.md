# Meteo Control Web

Interfaz web base para una estacion meteorologica con ESP32.

## Incluye

- Login de acceso (demo local con `localStorage`).
- Dashboard oscuro, moderno y minimalista.
- Tarjetas de metricas principales.
- Graficos de tendencias con Chart.js.
- Panel de alertas con umbrales configurables.
- Panel de ajustes (nombre de estacion y refresco).

## Ejecutar

Abre `index.html` en el navegador.

## Siguiente paso recomendado

Sustituir `fakeSensorData()` en `app.js` por datos reales desde ESP32 (HTTP/MQTT/WebSocket).
