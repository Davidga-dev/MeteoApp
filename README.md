# Meteo Control Web

Interfaz web para una estacion meteorologica con ESP32/Arduino.

## Incluye

- Login de acceso (demo local).
- Dashboard oscuro, moderno y minimalista.
- Tarjetas de metricas principales.
- Gráficos de tendencias con Chart.js.
- Panel de alertas con umbrales configurables.
- Panel de ajustes (nombre de estacion y refresco).
